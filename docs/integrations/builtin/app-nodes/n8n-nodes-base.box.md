# Box node <a href="#box-node" id="box-node"></a>

Use the Box node to automate work in Box, and integrate Box with other applications. n8n has built-in support for a wide range of Box features, including creating, copying, deleting, searching, uploading, and downloading files and folders.

On this page, you'll find a list of operations the Box node supports and links to more resources.

{% hint style="info" %}
**Credentials**

Refer to [Box credentials](../credentials/box.md) for guidance on setting up authentication.
{% endhint %}
 

## Operations <a href="#operations" id="operations"></a>

* File
    * Copy a file
    * Delete a file
    * Download a file
    * Get a file
    * Search files
    * Share a file
    * Upload a file
* Folder
    * Create a folder
    * Get a folder
    * Delete a folder
    * Search files
    * Share a folder
    * Update folder
