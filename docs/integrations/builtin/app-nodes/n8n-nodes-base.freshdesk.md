# Freshdesk node <a href="#freshdesk-node" id="freshdesk-node"></a>

Use the Freshdesk node to automate work in Freshdesk and integrate Freshdesk with other applications. n8n has built-in support for a wide range of Freshdesk features, including creating, updating, deleting, and getting contacts and tickets.

On this page, you'll find a list of operations the Freshdesk node supports and links to more resources.

{% hint style="info" %}
**Credentials**

Refer to [Freshdesk credentials](../credentials/freshdesk.md) for guidance on setting up authentication.
{% endhint %}

## Operations <a href="#operations" id="operations"></a>

* Contact
    * Create a new contact
    * Delete a contact
    * Get a contact
    * Get all contacts
    * Update a contact
* Ticket
    * Create a new ticket
    * Delete a ticket
    * Get a ticket
    * Get all tickets
    * Update a ticket
