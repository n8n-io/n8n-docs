# npm node <a href="#npm-node" id="npm-node"></a>

Use the npm node to automate work in npm, and integrate npm with other applications.

On this page, you'll find a list of operations the npm node supports and links to more resources.

{% hint style="info" %}
**Credentials**

Refer to [npm credentials](../credentials/npm.md) for guidance on setting up authentication.
{% endhint %}

## Operations <a href="#operations" id="operations"></a>

* Package
	* Get Package Metadata
	* Get Package Versions
	* Search for Packages
* Distribution Tag
	* Get All Tags
	* Update a Tag
