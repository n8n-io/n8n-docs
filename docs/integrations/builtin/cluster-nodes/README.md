## Root nodes <a href="#root-nodes" id="root-nodes"></a>

Each cluster starts with one [root node](https://app.gitbook.com/s/CxSeOtVxqqhfxMSac0AV/key-concept-glossary#root-node-n8n).

## Sub-nodes <a href="#sub-nodes" id="sub-nodes"></a>

Each root node can have one or more [sub-nodes](https://app.gitbook.com/s/CxSeOtVxqqhfxMSac0AV/key-concept-glossary#sub-node-n8n) attached to it.
