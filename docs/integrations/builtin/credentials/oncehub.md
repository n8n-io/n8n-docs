# OnceHub credentials

You can use these credentials to authenticate the following nodes:

- [OnceHub Trigger](/integrations/builtin/trigger-nodes/n8n-nodes-base.oncehubtrigger.md)

/// warning | Supported OnceHub plans
The OnceHub Trigger node relies on OnceHub webhooks. OnceHub only offers access to webhooks in their paid plans.
///

## Supported authentication methods

- API Key
