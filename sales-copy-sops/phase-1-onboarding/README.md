# Phase 1: Client Onboarding & Product Review

## Overview

This phase covers the initial intake process when a new client engages you for copywriting services. The goal is to understand the product at a fundamental level before any deep research or strategy work begins.

## Phase Objectives

1. Receive and catalog all client materials
2. Conduct initial product assessment
3. Identify product category and copy requirements
4. Create preliminary project brief
5. Prepare for kick-off call

## Required Inputs for This Phase

- Client contact information

## Quality Checkpoints

### Checkpoint 1.A: Materials Complete
- [ ] All promised materials received
- [ ] Materials organized and cataloged
- [ ] Access confirmed (if product access provided)
- [ ] Missing items identified and requested

### Checkpoint 1.B: Assessment Complete
- [ ] Product category clearly identified
- [ ] Product features documented
- [ ] Initial positioning understood
- [ ] Competitive context noted

### Checkpoint 1.C: Ready for Kick-off
- [ ] Preliminary brief complete
- [ ] Question bank prepared
- [ ] Call scheduled with client
- [ ] Recording/notes system ready
