## The SLA Framework (Story → Logic → Action)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        SLA FRAMEWORK                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   ┌───────────────────────────────────────────────────────────────┐     │
│   │                      STORY (PATHOS)                            │     │
│   │                                                                │     │
│   │  • Headline with curiosity + benefit + newness                │     │
│   │  • Opening story (relatable character with problem)           │     │
│   │  • Puzzling question (hooks them for the answer)              │     │
│   │                                                                │     │
│   └───────────────────────────────────────────────────────────────┘     │
│                              │                                           │
│                              ▼                                           │
│   ┌───────────────────────────────────────────────────────────────┐     │
│   │                      LOGIC (ETHOS + LOGOS)                     │     │
│   │                                                                │     │
│   │  • Establish credibility (who you are, why listen)            │     │
│   │  • Show failed alternatives (what they've tried)              │     │
│   │  • Reveal cracks in common solutions                          │     │
│   │  • Introduce Unique Mechanism (why THIS works)                │     │
│   │  • 3-5 Pillars/Principles/Secrets (the how)                   │     │
│   │  • Handle objections                                          │     │
│   │                                                                │     │
│   └───────────────────────────────────────────────────────────────┘     │
│                              │                                           │
│                              ▼                                           │
│   ┌───────────────────────────────────────────────────────────────┐     │
│   │                      ACTION                                    │     │
│   │                                                                │     │
│   │  • Transition to pitch                                        │     │
│   │  • Results/testimonials                                       │     │
│   │  • The crossroads (three choices)                             │     │
│   │  • The irresistible offer                                     │     │
│   │  • FAQ                                                        │     │
│   │  • Final CTA + P.S.                                           │     │
│   │                                                                │     │
│   └───────────────────────────────────────────────────────────────┘     │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## The Wormhole Email Framework

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    WORMHOLE EMAIL FRAMEWORK                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   STEP 1: SUBJECT LINE                                                   │
│   ├─ Curiosity + Self-Interest                                          │
│   └─ Teases the story or point                                          │
│                                                                          │
│   STEP 2: STORY                                                          │
│   ├─ Personal anecdote / observation / event                            │
│   └─ Something that happened to you, you saw, read, etc.                │
│                                                                          │
│   STEP 3: PIVOTAL POINT                                                  │
│   ├─ The lesson / insight from the story                                │
│   └─ Connection to your product/offer                                   │
│                                                                          │
│   STEP 4: TRANSITION                                                     │
│   ├─ Smooth bridge from point to pitch                                  │
│   └─ "And if you'd like to learn how to..."                            │
│                                                                          │
│   STEP 5: PITCH                                                          │
│   ├─ Sell the click, not the product                                   │
│   └─ Get them to the sales page                                         │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## Writing Best Practices

### Voice & Tone
- Write conversationally (like talking to a friend)
- Use "you" more than "we/I"
- Short sentences. Short paragraphs.
- One idea per paragraph
- Use customer language (from VOC)

### Formatting
- Lots of white space
- Bold key points
- Use bullets for lists
- Subheads every 300-400 words
- One sentence paragraphs for emphasis

### Emotional vs. Logical Balance
- Lead with emotion (story)
- Support with logic (proof)
- Return to emotion (transformation)
- Close with logic (offer value) + emotion (vision)
