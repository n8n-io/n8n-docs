# SOP 6.3: Ad Copy Creation

## Purpose
Create ad copy for various platforms (Facebook, Instagram, YouTube, Google) that stops the scroll, hooks attention, and drives clicks.

---

## Ad Copy Formula: HOOK → STORY → OFFER

```
HOOK (0-3 seconds)
├── Stop the scroll
├── Pattern interrupt
└── Speak to specific pain/desire

STORY/BODY (4-20 seconds)
├── Agitate problem OR
├── Tease solution OR
├── Share quick story

OFFER/CTA (Final)
├── What they get
├── Why act now
└── Clear next step
```

---

## Platform Specifications

### Facebook/Instagram

**Primary Text:** 125 characters visible, up to 3,000 total
**Headline:** 40 characters
**Description:** 30 characters

### YouTube

**Skippable:** Hook in first 5 seconds
**Non-skippable:** 15-30 seconds total

### Google Search

**Headlines:** 30 characters each (up to 15)
**Descriptions:** 90 characters each (up to 4)

---

## Hook Formulas (First 1-3 Seconds)

### Pain-Based Hooks
```
"Struggling with [specific pain]?"
"If [pain situation], this is for you..."
"Tired of [frustrating situation]?"
"Stop [common mistake] right now..."
```

### Curiosity Hooks
```
"I discovered something weird about [topic]..."
"The [adjective] truth about [topic]..."
"Here's why [common approach] doesn't work..."
"What if I told you [surprising claim]..."
```

### Result-Based Hooks
```
"How I [achieved result] in [timeframe]..."
"[Result] without [common obstacle]..."
"The exact [process] that got [person] to [result]..."
```

### Pattern Interrupt Hooks
```
"Wait - before you scroll..."
"I was wrong about [topic]..."
"Warning: This isn't for everyone..."
"I need to confess something..."
```

---

## Body Copy Templates

### Template 1: Problem-Solution
```
[HOOK]

I used to [struggle with same problem].

Then I discovered [mechanism/solution].

Now I [result achieved].

[CTA]
```

### Template 2: Story-Based
```
[HOOK]

[Brief story - 2-3 sentences]

The lesson? [Key insight]

That's why I created [solution].

[CTA]
```

### Template 3: Listicle
```
[HOOK]

Here's what most people get wrong about [topic]:

❌ [Mistake 1]
❌ [Mistake 2]
❌ [Mistake 3]

What works instead:

✅ [Solution]

[CTA]
```

### Template 4: Testimonial
```
[HOOK]

"[Powerful quote from testimonial]"
- [Name], [descriptor]

[They achieved this using our mechanism]

[CTA]
```

---

## CTA Templates

### Curiosity CTAs
```
"Click to see how →"
"Learn the method here →"
"Discover how →"
```

### Direct CTAs
```
"Get started now"
"Claim your spot"
"Start free trial"
```

### Urgency CTAs
```
"Get it before [deadline]"
"Only [X] spots left"
"Price increases [date]"
```

---

## Ad Variation Framework

Create multiple variations testing:

### Hook Variations (3-5)
- Pain hook
- Curiosity hook
- Result hook
- Question hook
- Pattern interrupt

### Body Variations (2-3)
- Problem-focused
- Solution-focused
- Story-focused

### CTA Variations (2-3)
- Curiosity CTA
- Direct CTA
- Benefit CTA

**Total Combinations:** Hook × Body × CTA = 15-45 variations

---

## Video Ad Script Template

### 15-30 Second Script
```
[0-3 sec] HOOK
"[Pattern interrupt or bold statement]"

[3-10 sec] PROBLEM
"If you're struggling with [pain], you're not alone."
"Most people try [solution] but it doesn't work because [reason]."

[10-20 sec] SOLUTION
"I discovered [mechanism]."
"It helped me [result]."

[20-30 sec] CTA
"Click below to learn how."
"[What they'll get/see]"
```

### 60-Second Script
```
[0-5 sec] HOOK
[5-15 sec] STORY/PROBLEM
[15-35 sec] SOLUTION/MECHANISM
[35-50 sec] PROOF/RESULTS
[50-60 sec] CTA/OFFER
```
